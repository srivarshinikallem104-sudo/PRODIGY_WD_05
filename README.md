🌦️ Weather Forecast Web Application

A modern, responsive weather forecasting web application built using HTML, CSS, and JavaScript, featuring a glassmorphism UI, real-time global weather data, and detailed hourly and daily forecasts.

📌 Project Overview

This project provides accurate and real-time weather information for locations worldwide.
It focuses on clean UI/UX, performance, and responsiveness, making it suitable for real-world use and frontend development practice.

✨ Features
🌍 Worldwide weather support using real-time API data
🔍 City search functionality
📍 Current location weather detection
⏱️ Hourly weather forecast
📅 5-day weather forecast
🌡️ Detailed metrics: temperature, humidity, wind speed, pressure, feels-like temperature
🎨 Modern glassmorphism design
📱 Fully responsive layout (desktop & laptop optimized)
⚡ Fast loading background with optimized assets
🛠️ Tech Stack
HTML5 – Application structure
CSS3 – Styling, glassmorphism UI, responsive layout
JavaScript (ES6) – Logic, API handling, dynamic updates
Weather API – Real-time global weather data
